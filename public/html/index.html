Examen No. 1

En base al código proporcionado, realizar las modificaciones necesarias para añadir las siguientes funcionalidades: 

    •	Creación de usuarios
    •	Modificación de usuarios

Para lograr este propósito, el código cuenta con una REST API (aquella realizada en clase), la cual ya posee el código asociado a los métodos POST y PUT. Adicionalmente, se ha proporcionado los formularios correspondientes en HTML para la creación/modificación de usuarios. Para su referencia, las vistas: layout e index ya se encuentran convertidas a formato pug.

Utilizando el esquema MVC, se deberán crear las vistas (contenido), controladores (rutas) y modelo (DB) para cumplir con el requerimiento.

Las implementaciones requeridas deberán cumplir todas las especificaciones solicitadas para obtener la máxima nota. Se deberá subir a la plataforma D2L en un archivo comprimido: el proyecto, con excepción de la carpeta node_modules.

Publicar el examen en heroku en una nueva aplicación, procurar nombrarla examen-{nombre.apellido}. Para lograrlo, deberán crear una nueva app en heroku.com con la convención de nombres anterior, y en el terminal local usar los siguientes comandos: 

    $ git init
    $ heroku git:remote -a examen-{nombre.apellido}
    $ git add *
    $ git commit -m "Comentario del cambio"
    $ git push heroku master
